# Лабораторная работа №2

## 📌 Тема
Основы NumPy: массивы и векторные операции

## 🎯 Цели работы
- Освоить работу с библиотекой NumPy для научных вычислений на Python
- Научиться создавать и манипулировать многомерными массивами
- Изучить векторные операции, матричные вычисления 
- Освоить статистический анализ данных и визуализацию результатов
- Применять стандарты качества кода: аннотации типов (PEP-484), документацию (PEP-257), стиль (PEP-8)

## 📝 Описание задачи и решение
Реализовать набор функций для работы с массивами NumPy, включая:
- Создание и преобразование массивов — arange, reshape, transpose
- Векторные операции — сложение, умножение на скаляр, скалярное произведение
- Матричные операции — умножение, определитель, обратная матрица, решение СЛАУ
- Статистический анализ — загрузка CSV, расчёт метрик, нормализация
- Визуализация — гистограммы, тепловые карты, линейные графики

## ✅ Выводы
В ходе лабораторной работы были освоены ключевые возможности библиотеки NumPy: создание и преобразование массивов, векторные и матричные операции, статистический анализ. Реализованные функции проходят все тесты, код аннотирован, документирован и оформлен в соответствии с общепринятыми стандартами
